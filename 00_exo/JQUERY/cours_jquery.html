JQuery, une librairie Javascript pour gagner en productivité
'Write less, Do more'

Version actuelle : v3.3.1

JQuary c'est une bibliothèque javascript, ou un framework. C'est une boite à outils de blocs de code réutilisable. Cette librarie est rapide, petite donc légère et riche en fonctionnalités.

Elle simplifie considérablement la navigation et la manipulation dans le DOM des pages HTML, le gestion des événements, l'animation et Ajax(Asynchronous Javascript And XML) grace à une API (application programming interface)facile à utiliser et qui fonction sur beaucoup de navigateurs and qui est déjà reé testée.

JQuery est elle-meme codée en Javascript.

Polyvalence et extensibilité, JQuery a changé la façon de travailler de millions de développeurs.

  • Fonctions toutes faites, optimisées, qu'on utilisera souvent : appel Ajax, boucles, animations, toggle
  • Résond bcp de probmèmes de comptabilité navigateurs.
  • JQuery a des polifills intégrés
  • Une lecture de code plus simple car les probléme sont gérés par la librairie

Attention: JQuery n'est pas à implique en pansement sur un code sale (Tag Soup....)

JQuery a des millions de plugins additionnels
