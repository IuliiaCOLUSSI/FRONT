Sélectionner en JQuery

Le coeur de JQuery se trouve dans la sélection d'éléments pour ensuit exécuter des méthodes dessus.

$(selecteur) accepte en argument tout sélecteur CSS, JQuery nous renvoie ensuite une collection d'un ou pluiseurs éléments DOM enveloppés dans un objet JQuery.

$('div') séléctionne tous les dives $('.titre') séléctionne tous les éléments qui ont la classe = "titre" $('#boite') selectionne l'unique éléments qui l'id = "boite"

Cette enveloppe JQuery est celle qui donne accès à tous les fonctionnalitès de JQuery

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher les éléments
  • Les animer etc.

Tous les éléments sélectionnés en JS pur (document.getElementById, querySelector etc) sont les éléments DOM et tous sélectionnés avec $('') sont des éléments DOM dans une enveloppe JQuery

JQuery dispose de pleuseurs méthodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants direct
  • find() parcourt les descedants
  • next() renvoie les voisin suivant
  • prev() renvoie les voisin précédent
  • parent() renvoie le parent
  • eq() renvoie, parmi une liste d'éléments, celui qui se trouve à l'index fourni

les getters et setters, ajouter, retirer des classes et savoir si on a déjà des classes

  • addClass() : ajoute une class
  • removeClass() : retire une class
  • toggleClass() : interrupteur ajout / retire une class
  • hasClass() : tester l'existance d'une class
  • is() : tester si on a pleuseurs class

    click : au clic sur l'élément

    mouseleave : lorsque la souris sort de cette zone

    keydown : une touche de clavier enfoncée
    keyup : une touche de clavier a été relanchée
  • window (la fenetre du navigateur)
    scroll : défilment de la fenetre
    resize : redimensionnement de la fenetre
