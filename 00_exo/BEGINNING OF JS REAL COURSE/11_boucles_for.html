Les boucles

Les instructions FOR et WHILE qui sont deux types de boucles les plus utilisées en js, servent à répéter pluiseurs lignes de code

La boucle FOR

FOR prend 3 paramètres encadrés dans des parenthèses, suivi enter accolades des instructions à éxécuter autant de fois que la boucle l'indique. Les paramètres sont :

  • Initialisation : ici on déclare la variable qui servira de compteur
  • Condition : Tant que cette condition est vraie la boucle s'éxécutera
  • Expression finale : expression qui est éxécutée à chaque fin de boucle
for(initialisation; condition; incrémentation){ instruction1; instruction2; }
